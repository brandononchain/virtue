# Skill: Lighting Design

## Purpose
Simulates cinematic lighting setups.

## Responsibilities

- Key / fill / back light balance
- Motivated lighting sources
- Film-grade color grading
- Realistic shadow physics

## Supported Styles

- Noir
- Studio portrait
- Natural daylight
- Neon cyberpunk
- Golden hour
- Horror low-key lighting

## Output

Lighting rig configuration used during render generation.
