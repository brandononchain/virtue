# Skill: Character Performance

## Purpose
Generates believable human or character motion.

## Responsibilities

- Facial expressions
- Body language
- Actor blocking
- Emotion fidelity
- Dialogue timing

## Performance Layers

- Micro expressions
- Gestures
- Eye focus
- Movement pacing
- Interaction with environment

## Example

Prompt:
"Detective realizes the suspect is lying"

Generated behaviors:
- subtle eyebrow raise
- pause before speaking
- slight lean forward
- narrowed eyes
